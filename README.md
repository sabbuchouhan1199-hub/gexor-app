# Gexor

Gexor is a provider-independent AI runtime platform designed to preserve and manage the user's context, memory, knowledge, and workspace independently of any individual AI provider.

## Project status

This repository is being rebuilt from a clean baseline using a manual, verification-first development workflow.

The application is not yet implemented.

## Development principles

- Build one verified vertical slice at a time.
- Understand every file and command before adding it.
- Keep secrets out of source control.
- Run validation before every commit.
- Do not allow autonomous agents to make uncontrolled repository changes.

## Documentation

The approved Gexor product and architecture documents are maintained separately in the `gexor-docs` repository.

Those documents are the source of truth for product scope and system behavior.
